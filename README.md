# ⚡ Praveen Murugan — Cyberpunk Portfolio

> **Live:** [0xpr4v33n.github.io/pfo](https://0xpr4v33n.github.io/pfo/)

A scroll-reactive, fully-animated personal portfolio for **Praveenkumar Murugan** — Senior Cybersecurity Consultant @ LTIMindtree. Built as a single-page static site with zero dependencies, zero build step, zero framework.

## ✨ Features

### 🎨 Visual & Theme
- Cyberpunk 2077 palette (yellow / cyan / magenta)
- Custom cyberpunk cursor (yellow crosshair → cyan/red reticle on clickables)
- Interactive **network particle canvas** (mouse-repel, scroll-reactive color)
- **Cyber matrix rain** with CVE / IP / hex / HTTP-code fragments
- Animated **SVG circuit-board dividers** between sections
- **Aurora gradients**, vignette, scanlines, noise overlay
- **Glitch effects** on hero name & card titles (RGB-split)
- **ASCII topology** previews on each project card

### ⚡ Interactive
- **Mouse-follow gradient glow** on every card
- **Edge-runner streak** lapping each card border on hover
- **Cycling neon border** animation (yellow → magenta → cyan)
- **Particle burst** on every button/link click
- **Scroll-spy** with animated nav underline
- **Project filter chips** (all / migration / automation / etc.)
- **Eased counter** animation on stats
- **Animated timeline** with pulsing diamonds + flowing line + staggered bullets

### 🔧 Cyberpunk Pro Features
- 🛡️ **Security consent banner** (gate before page load)
- 💾 **Terminal-style boot loader** (12 typed lines + progress bar)
- 📡 **Live HUD overlay** (uptime, scroll %, jittering GPS coords, node status)
- 🔊 **Procedural ambient audio** (Web Audio API · synthesized cyberpunk drone)
- 🖱️ **Custom right-click menu**
- 🎮 **Konami code easter egg** (↑↑↓↓←→←→BA)
- 📝 **Text scramble decode** on section titles
- ⬆️ **Back-to-top button**
- 📋 **Copy-to-clipboard** on email/phone
- 📊 **Live GitHub stats** on project cards (via GitHub API)
- 📄 **Resume preview modal** with inline PDF viewer
- ✉️ **Working contact form** (Formspree-ready)
- 🪟 **Cert PDF hover preview** with fullscreen modal

### 🚀 SEO / Performance / Accessibility
- **Open Graph + Twitter Cards** with custom preview image
- **JSON-LD structured data** (Schema.org Person)
- **Inline SVG favicon** + PWA manifest
- **Skip-to-content** link for keyboard a11y
- **Page-visibility pause** (canvases freeze when tab inactive)
- Fully responsive (4K, desktop, tablet, mobile)
- Honors `prefers-reduced-motion`

## 🛠️ Tech Stack

- **Pure HTML5 / CSS3 / Vanilla JavaScript** (ES6+)
- **Fonts:** Inter, JetBrains Mono, Orbitron
- **Canvas API** — matrix rain & network particles
- **Web Audio API** — synthesized ambient drone
- **IntersectionObserver** — scroll-spy & reveal animations
- **Web Animations API** — particle bursts
- **GitHub REST API** — live repo stats

**Zero dependencies. Zero build step. Zero framework lock-in.**
